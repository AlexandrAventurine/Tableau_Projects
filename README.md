# 🏡 Airbnb Data Analysis (Tableau Project)

## 📌 Project Overview
This project analyzes Airbnb listings in Washington to uncover pricing trends, demand patterns, and key business insights.

## 🛠 Tools & Technologies
- Tableau
- Data Visualization
- Exploratory Data Analysis (EDA)

## 📂 Dataset
Airbnb dataset containing listing prices, locations, availability, and property details.

## 📊 Dashboard Features

### 🔹 Pricing Analysis
- Price distribution across neighborhoods
- Average price per location

### 🔹 Availability Insights
- Seasonal trends
- Occupancy patterns

### 🔹 Property Analysis
- Property types comparison
- Popular listing categories

### 🔹 Location Insights
- High-demand areas
- Revenue potential by region

## 🎯 Key Insights
- Pricing varies significantly by location
- Certain neighborhoods dominate demand
- Property type impacts revenue potential

## 💡 Key Skills Demonstrated
- Data Visualization in Tableau
- Interactive Dashboard Creation
- Business-Oriented Analysis
- Data Exploration

## 🚀 Outcome
A visual analytics solution for understanding Airbnb market trends.
